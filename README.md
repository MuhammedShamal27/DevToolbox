# DevToolbox

A modular and extensible platform for developer tools, built with Next.js (App Router), TypeScript, and Tailwind CSS. The design is clean, minimalist, and inspired by highly polished UI frameworks.

## Features

- **Base64 Converter**: Encode and decode strings.
- **Password Generator**: Generate secure, random passwords.
- **YAML Diff Checker**: Compare two YAML configurations side-by-side.
- **Theme Support**: Seamless light and pure black dark mode toggling.
- **Modular Architecture**: Easy to add new tools through a simple registry system.

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/MuhammedShamal27/DevToolbox.git
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
